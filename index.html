B1-Forum

Forum

Création d'un site web accessible par navigateur et qui a pour but d’être un espace de discussion orienté questions / réponses.

Nous pouvons retrouver toute les fonctionnalités d'un forum basique tel que le fait de pouvoir s’inscrire et se connecter, créer des topics, poster un message, modifier ou supprimer un topic ou un message, modération...

Tout ça géré grâce à une base de donnée SQL regroupant toutes les informations utilisateurs.

B1-Groupie-Traker

Groupie Traker

Un projet mettant en avant les API (Application Programming Interface) facilitant la création et l'intégration de logiciels d'applications.

Le but est simple se servir d'une API comme base de donnée pour créer un site web dynamique répertoriant toutes les informations de cette dernière.

Avec mon duo nous avons choisit de créer notre propre API (REST) sur le jeu "Monster Hunter World" que nous avons hébergé sur Github.

B1-Hacky'Nov

Hacky'Nov

Hacky'Nov est une startup événementielle proposant un CTF (Capture The Flag) permettant de réunir les étudiants en informatique et le grand publique sur le vaste domaine de la cybersécurité.

Mes coéquipier et moi avons proposé des challenges de toutes catégories, afin que les participants puissent en résoudre le maximum pour gagner la compétition !

Des entreprises partenaires (engIT, AIX YNOV CAMPUS, Root-Me Pro, Synacktiv, SQUAD Conseil et Expertises, Atos Digital Security, SAP Labs France et Credypto) nous on accompagné tout le long de l'événement en organisant des conférences pour sensibiliser les participants.

B1-Hangman-Web

Hangman Web

Qui ne connais pas le jeu du pendu ? Personne, bien évidemment.

Un de mes premiers projets, la réalisation d'un hangman web répondant à des requêtes via une communication client/serveur utilisant le protocole HTTP.

Mon groupe et moi avons choisit comme thème le jeu "Undertale" pour renouveller le concept. Ce dernier est lancé en local et fonctionne dynamiquement grâce à un fichier .txt lui sert de base de donnée pour choisir un mot aléatoire au début de chaque partie.

B1-TP-Web

TP Web

Mon premier site web ayant pour thème les mangas et les animés.

Touchant aussi bien les enfants, les adultes et même les personnes âgées… la culture japonaise se retrouve partout dans le monde, ainsi nous Européen avons découvert les mangas et les animés, c'est pour cette raison que nous avons créer "Kitsune".

Un site web répertoriant les grands sites de streaming d’animé, mettant à disposition des fonds d’écran full HD pour satisfaire vos désirs de personnalisation ainsi que nos avis sur les animés et mangas que nous avons lu.

B1-RPG

RPG

Mon tout premier projet en informatique : un RPG réalisé entièrement en GO fonctionnant avec une console de commande.

Même si ce n'est pas un grand projet, ce dernier fût une découverte pour moi, me permettant de réaliser mes premières lignes de code. Avec mon duo nous avons choisit d'écrire une histoire resemblant à "Sword Art Online".

Selection de la classe, marchand, forgerons, monstre, item unique, inventaire, ecpérience ou encore boss s'inviteront dans ce petit jeu permettant de passer le temps.

B1-Infra-SI

Infra SI

Mise en place d’un SI d’une petite entreprise. Le but étant de proposer une architecture comprenant : Un parefeu (avec DHCP, DNS, filtrage réseau, routage…), 2 clients (1 windows et 1 linux), 1 serveur linux (sauvegarde), 1 serveur linux (hébergement site web) avec accès restreint.

Nous avons réaliser une maquette du site web (servant de site vitrine pour l'entreprise), et avons établie un schémat réseau, plan d'adressage, tableau de filtrage, sauvegarde automatique, restauration et documentation nécéssaire à la mise en place du projet.

B1-JavaScript

JavaScript

7 exercices organisés sous forme de jeu pour mieux appréhender le JavaScript.

La création d'un carrousel, animation, quizz, API météo, machine à sous, interface de vente et un jeu avec chemin aléatoire

B1-Challenge-48h

Challenge 48h

Un challenge d"un durée de 48h au sein de l’établissement AIX YNOV CAMPUS pour répondre la une problématique suivante :

Le campus souhaite améliorer sa digitalisation en ayant un système de gestion des téléviseurs. Le but est de proposer une solution clé en main pour l’administration ainsi que le budget nécessaire pour la réalisation de celle-ci.

Notre solution : utiliser des raspberrys Pi connectés à un/des routeur(s) et créer un logiciel de gestion système des raspberrys avec utilisateur et administrateur pour afficher du contenus a distance.
